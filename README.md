# Stremio Addon SDK

Community Stremio Addon SDK for modern-day TypeScript developers.

## Setup

### Prerequisites

- Node.js (latest LTS recommended)
- Corepack enabled

### Installation

1. Enable Corepack (if not already enabled):

   ```bash
   corepack enable
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

### Development

Build all packages:

```bash
pnpm build
```

Format code:

```bash
pnpm format:fix
```
